# Raw Idea: Projection Aggregation & Analysis System

## User's Description

A system to aggregate player projections from multiple sources (ETR, DK, etc.) with robust player identification (DK Player ID), contest-specific ownership data, floor/ceiling ranges, and analytical tools to evaluate projection accuracy and build differentiated DFS lineups for cash vs tournament play.
